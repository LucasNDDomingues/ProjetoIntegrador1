# Projeto Integrador - Introdução ao Java

Repositório da disciplina Projeto Integrador com foco em introdução à linguagem Java, contendo exercícios e práticas desenvolvidas em aula.

## 🎯 Objetivo
Desenvolver a base em lógica de programação e estruturas fundamentais utilizando Java.

## 📚 Conteúdo
- Estruturas condicionais (if/else)
- Estruturas de repetição (while)
- Entrada de dados com Scanner
- Geração de números aleatórios com Random
- Exercícios práticos

## 🧠 Exemplo
Implementação do jogo "Acerte o Número", onde o usuário tenta adivinhar um número gerado aleatoriamente.


## 🛠️ Tecnologias
- Java

## ▶️ Como executar
1. Compile o arquivo `.java`
2. Execute a classe principal

## 📌 Observações
Projeto com finalidade educacional para prática dos fundamentos de programação.
